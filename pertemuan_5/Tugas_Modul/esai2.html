Tipe Data dalam Array JavaScript

Hasil:

Array dan tipe data akan ditampilkan di sini

Penjelasan:

Array dalam JavaScript dapat menyimpan berbagai jenis tipe data, termasuk:

  • Number - angka (contoh: 42, 3.14)
  • String - teks (contoh: "Hello World")
  • Boolean - nilai kebenaran (true/false)
  • Null - nilai kosong yang disengaja
  • Undefined - variabel yang belum diberi nilai
  • Object - kumpulan properti dan nilai
  • Array - kumpulan nilai (array dalam array)
  • Function - blok kode yang dapat dipanggil
